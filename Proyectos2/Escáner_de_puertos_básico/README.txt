El ejecutable se úbica en la carpeta nombrada dist.
1- Ingrese una dirección IP para escánear.
2- Ingrese el número de puerto donde iniciará el escáneo.
3- Ingrese el número de puerto en donde finalizará el escáneo.
4- Presionar iniciar escáneo.
5- Se creará un archivo de texto con el informe de los puertos escáneados.
